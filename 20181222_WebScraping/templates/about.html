About

This assignment entailed building a web application that scrapes various websites for data related to the Mission to Mars theme. The objective was to display the information from each source in a single HTML page.


The techniques and tools used to compile this information included:

  • Python Modules such as Pandas, Time, Beautiful Soup, Splinter, Requests, Pymongo
  • Jupyter Notebook
  • Flask Framework
  • MongoDB
  • HTML/CSS and the Bootstrap Framework

    • A Jupyter notebook was leveraged to test the scraping and analysis. Several websites were scraped in order to gather the following information:

  • NASA Mars News
  • JPL Mars Space Images (Featured Image)
  • Mars Weather
  • Mars Facts
  • Mars Hemispheres

    • Once the scraping was complete, the information obtained was stored in a Python dictionary. Next, I converted the Jupyter notebook into a Python script that could be called via a function. I used the Flask framework to create a connection to Mongo database that was used to store the scraped information. Additionally, I created routes to tell Flask what URL should trigger the functions. For example, the root "/" route was created in order to query my Mongo database and pass the scraped information about Mars into an HTML template used to display the data. Another route, "/scrape" route triggered the function that runs the scraping script.
